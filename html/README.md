# MyCalc_HTML
HTMLで作った電卓

## 使用した言語
|使用言語       |使用目的                       |
|:--------------|:-----------------------------|
|HTML           |オブジェクトの配置             |
|SCSS,CSS       |見た目の補正                   |
|JS             |実際の演算処理                 |

## 主なクラス
|クラス名            |使用目的                    |
|:-------------------|:--------------------------|
|str_box             |計算結果の出力              |
|grid_box            |ボタンのグリッド配置        |
