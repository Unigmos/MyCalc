## Eval関数について
本プログラムでは複数の言語でeval関数を用いています。<br>
セキュリティ面で問題とされ、evil関数とも言われていますが、ローカル環境のみの稼働&文字列の演算をeval関数を用いないととんでもないコード量になってしまうことを考慮しeval関数を用いています。
