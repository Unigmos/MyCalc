# MyCalc_Python

## 使用したライブラリ
|使用ライブラリ       |使用目的                       |
|:-------------------|:-----------------------------|
|tkinter             |デスクトップアプリ用ライブラリ  |

|変数名              |使用目的                    |
|:-------------------|:--------------------------|

|set_decimal_window  |小数点桁数設定子ウィンドウ   |
